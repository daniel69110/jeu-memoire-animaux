## 🎮 Comment jouer

1. **Cliquez sur une carte** pour la retourner
2. **Trouvez les paires** d'animaux identiques
3. **Moins de temps = meilleur score** !

## 🚀 Fonctionnalités

- Animations fluides CSS
- Compteur de temps et mouvements
- Thème animaux mignons
- Responsive mobile/tablette
- **0 dépendance** - pure JS

## 📱 Installation

**Aucune installation requise !** Ouvrez simplement `index.html` dans votre navigateur.

```bash
git clone https://github.com/daniel69110/jeu-memoire-animaux.git
cd jeu-memoire-animaux
# Ouvrir index.html
```
